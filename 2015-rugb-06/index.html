Eval & Apply

(EVAL (> 9 (- 10 2)))
=> (APPLY > 9 (EVAL (- 10 2)))
=> (APPLY > 9 (APPLY - (10 2)))
=> (APPLY > 9 8)
==> true

Eval: Lambda

(def add (lambda (x y) (+ x y)))

 

binding:
  add: {bind: b, args:.. , body:.. }

 

(add 3 4)
=> (APPLY {bind: b, ...} 3 4)

APPLY: (lambda (x y) (+ x y))

Step 1. bind arguments

(APPLY add 3 4)

binding:
  x: 3
  y: 4

APPLY: (lambda (x y) (+ x y))

Step 2. evaluate body

binding:
  x: 3
  y: 4

(EVAL (+ x y))
=> (APPLY + (EVAL x) (EVAL y))
=> (APPLY + 3 4)
==> 7

Recursion

(def map
     (lambda (f l)
       (if (nil? l)
           nil
         (cons (f (car l)) (map f (cdr l))))))

(map (lambda (x) (* x x)) '(1 2 3))
==> (2 4 9)
